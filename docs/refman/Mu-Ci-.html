移動:

カーソルを使用するか、"h" で左、"j" で下、"k" で上、"l" で右に移動できます。

ヘルプの終了:

":q<Enter>" とタイプします。

Vim の終了:

":qa!<Enter>" とします (注意！變更はすべて破毀されます)。

項目へジャンプ:

タグ (例: |bars|) にカーソルを合はせ、CTRL-] を押します。

マウスでジャンプ:

":set mouse=a" と入力してマウスを有效にしてください (xterm または GUI で利用可)。タグ (例: |bars|) をマウスの左ボタンでダブルクリックすると、項目にジャンプできます。

元の場所に戾る:

CTRL-O を押します。繰り返すと更に戾れます。

項目を指定する:

|:help| コマンドに項目を指定することで、その項目を直ちに表示できます。修飾子を先行入力することで、調べたい項目の種類を指定することもできます:

種類修飾子例
ノーマルモードコマンド:help x
ビジュアルモードコマンドv_:help v_u
插入モードコマンドi_:help i_<Esc>
コマンドラインコマンド::help :quit
コマンドライン編輯c_:help c_<Del>
Vim の起動引數-:help -r
オプション’:help 'textwidth'
正規表現/:help /[

その他の種類については |help-summary| を參照してください。

ヘルプを檢索する:

":help word" とタイプしてから CTRL-D を押すと "word" にマッチするヘルプ項目の一覽が表示されます。もしくは ":helpgrep word" を使つてください。|:helpgrep|

Bars example

今、あなたはマウスのダブルクリックか CTRL-] でジャンプしてきました。CTRL-T、CTRL-O、g<RightMouse>、<C-RightMouse> のどれかで元の場所に戾れます。

Note:
タグは | で圍まれてゐますが、その文字は、カラー表示が有效になつてゐる場合は表示されません。讀みやすさのためです。

CTRL-] はどの單語の上でも使へます。| で圍まれてゐない單語でも大丈夫です。選擇された單語がヘルプから檢索されます。特にシングルクォートで圍まれたオプション名 (例: ’compatible’)。
