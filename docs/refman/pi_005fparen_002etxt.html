                  VIMリファレンスマニュアル    by Bram Moolenaar

對應するカッコの强調表示

このプラグインは標準プラグイン |standard-plugin| です。’compatible’ がオフのときだけ利用できます。

このプラグインが讀み込まれないやうにするには變數 "loaded_matchparen" を設定してください: 

:let loaded_matchparen = 1

このプラグインは自動コマンドイベントの CursorMoved, CursorMovedI, WinEnter を使つて强調表示を再設定します。

このプラグインの機能を無效にするには次のコマンドを使ひます: 

:NoMatchParen

機能を有效に戾すには次のコマンドを使ひます: 

:DoMatchParen

强調表示には MatchParen が使はれます。":highlight" コマンドを使つて好きな色を設定できます。例: 

:hi MatchParen ctermbg=blue guibg=lightblue

オプション ’matchpairs’ に設定されてゐる文字が强調表示されます。このオプションを設定することで好きな文字を强調表示できます。Note: 設定しても機能しない文字もあります。例へば、シングルクォートやダブルクォートのペアなどは、始點と終點が同じなので强調表示できません。

對應するカッコを探すときには構文强調の屬性が使はれます。カーソル位置の文字が、文字列やコメント以外のときは、文字列やコメントの中は無視されます。名前に "string" や "comment" が含まれてゐる構文强調アイテムはすべて、文字列やコメントとして扱はれます。

カーソルの移動が遲くならないやうに、對應するカッコの檢索には次の制限があります:

  • ウィンドウに表示されてゐる範圍だけ檢索する。
  • 閉ぢた折り疊みがあると處理に時閒がかかるので、カーソル位置から 100 行以內の範圍だけ檢索する。
  • 長い行が構文强調表示されてゐると處理に時閒がかかるので、カーソル位置から ’synmaxcol’ の 2 倍の範圍の文字だけ檢索する。
  • 300 ミリ秒でタイムアウト (插入モードでは 60 ミリ秒)。これは變數の g:matchparen_timeout と g:matchparen_insert_timeout を設定することで變更できる。バッファ每に設定する場合は b:matchparen_timeout と b:matchparen_insert_timeout を設定する。

|%| コマンドの動作を改善したい場合は matchit プラグインを使つてください。|matchit-install| 參照。このプラグインを使ふとコメント內のカッコをスキップすることもできます。これは matchparen プラグインとの關聯性はありません。matchit プラグインは matchparen とは別のメカニズムを使つてゐます。
