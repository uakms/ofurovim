Vim ユーザーの感想

*quotes.txt*    For Vim バージョン 8.1.  Last change: 2018 Mar 29


		  VIMリファレンスマニュアル    by Bram Moolenaar

ニュースやメールから集めた、Vim に關する素晴らしい引用を紹介します。

vim (vim) 名詞 - 溢れんばかりのバイタリティとエネルギー。 [ラテン語, 『力』の直接目的格, 力] (辭書)

Vim は vi より大變優れてゐるので、私の古い vi の多くの偉大な :map がすぐに廢れてしまひました！ (Tony Nugent, オーストラリア)

Windows の GUI 志向の强い影響で、Vi を使ふ人々は實は人閒の姿をしたエイリアンなんぢやないかと思つてゐました。でも一度試してみるとそのパワーに病みつきになつて、今では Vim 以外のソフトウェアで Vim のやうにキータイプするまでになつてしまひました。これが、テキストを編輯するあらゆるアプリケーションに Vim を埋め込まれて欲しいと願ふ理由です。(José Fonseca)

私は 12 年來の emacs ユーザーでしたが、相次ぐ互換性のないバージョンアップや、役立たずの追加パッケージ群、一貫性のない操作方法などに嫌氣がさして 1 年前に Vim に乘り換へました。まさに生涯最高の選擇の 1 つでした。 (Joel Burton)

資料の作成や修正に多くのプログラムを使用しましたが、編輯のほとんどは GNU-Linux (RedHat 4.2) 上の Vim のバージョン 4.5 と 5.0 で行はれました。(Arnold Robbins, イスラエル, "Learning the Vi editor" の著者)

私は幾つものオープンソフトを使ひ、さらに多くのソフトを見てきましたが、その中でも Vim は大變使ひやすく、とても完成度が高く、linux kernel に次ぐ最高の作品です。(Peter Jay Salzman)

SourceForge の「全體が」Vim とその PHP 用のイカシた構文强調を使つて書かれてゐる、といふことは大して重要ではありません。大事なのは SF.net の技術スタッフ全員が Vim を使つてゐるといふことであり、そして貴方が Vim を使ふことを大いに歡迎してゐるといふことです！(Tim Perdue)

Vim は私の七つ道具の 1 つで、代用品がありません。なんてすばらしい作品なんだらう！(Biju Chacko)

私の元カノは emacs に走りました。もちろん、彼女とはうまくいきませんでした。(Geoffrey Mann)

魚が水のことを意識しないと思はれるのと同樣に、私はほとんど Vim について意識しません。それは全ての中にある環境です。私は相當に忙しいシステム管理者で、多くの異なるプラットフォームを扱つてゐます。Vim は 私のシステム全てに共通して存在する唯一のもので、「時々クラッシュする」といふこともない唯一のものです。新しいシステムに Vim が入つてゐなかつたときには、私は眞つ先にインストールします。どの環境でもまつたく信賴のおける同じツールを使ふことができることは素晴らしく、おかげで Vim については無視することができ、その他のものについて考へることができるのです。(Pete Schaeffer)

最近ノキアの端末を通して telnet 經由で Vim を起動することに成功しました、現在 Palm Pilot でも素晴らしく動いたことを報告できます。(Allan Kelly, スコットランド)

Bram! ’VIM’ といふなんてものすごい仕事をしたものです！使へば使ふほど感動します (私は古い ’vi’ のダイハードで 1970 年代後半の初期の ’emacs’ から使ひ始め、1983 年に初期の UNIX で ’vi’ を見つけることで救はれました)。私の考へでは、標準のトップエディタとして ’VIM’ が ’emacs’ に置き換はる時です。(Bo Thide’, スウェーデン)

私も Vim を深く愛し使つてゐます。(Larry Wall {譯注: Perlの作者})

Vi はまるでフェラーリです。初心者にはまるで暴れ馬ですが、いつたんコツさえ摑んでしまへば、非常に小さくパワフルかつ高速です！(不明) Vim はまるでフェラーリの新モデルです。撥音もそつくり - "VIIIIIIMMM!"。(Stephen Riehm, ドイツ)

Schon bei Nutzung eines Bruchteils der Vim-Funktionen wird der Benutzer recht schnell die Vorzuege dieses Editors kennen- und schaetzenlernen. 飜譯: Vim の機能をほんの少し使ふだけでも、ユーザーはすぐに使ふことができこのエディタの强みを正しく評價するでせう。(Garry Glendown, conclusion of an article on Vim in iX magazine 9/1998)

つい最近 Vi についてのオライリーの本を手に入れました (Vim についても深く言及されてゐました)。このアプリケーションがあまりにパワフルで驚愕しました。(Jeffrey Rankin)

このガイドは Windows 9.x 上の gvim で書いてゐます。これは神が裸の女性を創造して以來の最高傑作なんぢやないでせうか。(Michael DiBernardo)

少年よ、私は私自身 Vim についてほとんど全てのことを知つてゐると考へてゐました。しかしいつもオンラインドキュメントを眺めるたびに、それまで知らなかつた Vim の機能の小さな、しかしクールな面を見つけるのです！これは私が見た中でも (多くの商業製品と比べてさへも) 最高のドキュメントであると言はねばなりません。(Gautam Mudunuri)

Vim 4.5 は本當に素晴らしい (ファンタスティックな) エディタです。Vim にはものすごぉぉぉぉい多くの機能があり、更に重要なことには初期設定が素晴らしいので本當に何の變更も無しに使ふことができるのです！！Vim 作者への私の驚きと感謝の念はどんな言葉をもつても語り盡くせません。この先もさうであつて。(Vikas, アメリカ合衆國)

人々が他の Vi エディタを Vim のクローンと呼ぶやうになるまでどのくらゐかかるのだらう？ (Darren Hiebert)

"Perfect Programmer’s Editor" のスレッドにある [make のエラーに基づいて自動的にファイルの位置へ飛ぶ] といふのを讀んで、Vim は既にそれをサポートしてゐるので嬉しかつた。(Brendan Macmillan, オーストラリア)

Vim (5.0) を發見して多くの知人とそれについて話をしました。Vim は Vi の新世紀 (Millenium) を象徵するんだ、と彼らに言ひ聞かせました。本當にありがたう！(Matt F. Valentine)

"vi" は今から "Vim Imitation (Vim のものまね)" と呼ばれるべきで、逆ではないと考へます {譯注: Vim はもともと Vi IMitation に端を發してゐます}。(Rungun Ramanathan)

Vim の法則 {譯注: 數學の公式風に！}:
プログラム P の考へ得る限りの動作空閒 B の中で各々の要素を b とした時、プログラム P の全ユーザー空閒Uのうち少なくとも 1 人のユーザー u が、ある動作 b を許された動作空閒 B’ に加へるやうに要求するまでにかかる時閒は有限です。(このとき B’ は B’ <= B となります) 言ひ換へれば: 遲かれ早かれ全員がオプションとして全てを欲しがるのです。(Negri)

あたらしい計算機プラットフォームに移る時はいつも、決まつてまづ最初にやるのは Vim を移植することです。後に configure を使つた時にコンパイルの簡單さにビックリしました。(A.M. Sabuncu, トルコ)

オプションが本當に優秀でとてもパワフルです。(Anish Maharaj)

ユーザーインターフェイスのデザインの入つてゐる春、そしてブティックからの便りは「80x24 テキスト專用モードは復讐と共に戾ります！」。Vi エディタのクローンである Vim は、5.0 のパステル調のシンタックスハイライトのまばゆいばかりの見た目と共に、3 月のデスクトップに突然現れました。執拗なそしてカスタマイズ可能な、Vim はその傳統的な Vi シングルキーマクロ集の解釋で驚かせます。{譯注: 原文が難解でまつたく自信無} http://www.ntk.net/index.cgi?back=archive98/now0327.txt&line=179#l

Vim 5 の衝擊をどうしても傳へたかつたのです！シンタックスハイライティング: これ無しでどうやつて生きろと？！人類最高のエディタを作つてくれてありがたう！(Mun Johl, アメリカ合衆國)

重ね重ね Vim には感謝します。Linux で每日使つてゐます。(Eric Foster-Johnson, "UNIX Programming Tools"の著者)

史上最高のエディタです (Stuart Woolford)

Vim の素晴らしい機能のほとんどを少なくとも 1 度は、ものによつてはもう何度も頻繁に使つてきました。正直に白狀します、どれ 1 つ缺けても生きていけません。vi を使つてゐた頃の自分と生產性を比べると、優に倍にはなつてゐます。(Sitaram Chamarty)

Vim を愛してゐます。信じられません。最初の子供に Vimberly と名づけました。(Jose Unpingco, アメリカ合衆國)

ヒント: "Vim" は "vi improved (vi改)" といふほうが良い！(Sven Guckes, ドイツ)

每日 Vim を使つてゐます。他のどんなプログラムよりも Vim と共に過ごす時閒が長い…。存在する vi クローンの中で最高です。とてもすごい。(Craig Sanders, オーストラリア)

その無限アンドゥ/リドゥで私を幾度となく悲しみから救つてくれました--- Vim を使用するやうに强く薦めます。(Terry Brown)

私の考へうるなかで、この惑星で最良のテキストエディタを書いてくれたことにたいへん感謝します。次に猫を飼ふときは "Vim" と命名したい。(Bob Sheehan, アメリカ合衆國)

:set all とタイプすればオプションで畫面が埋め盡くされます。畫面全體のものが設定か非設定かされてゐます。wrapmargin や modelines や showmode といつた古い友人たちも見つけましたが、新しい友人たちで畫面は埋め盡くされてゐました。私は彼らを愛してゐます！！私は Vim を愛してゐます！！このエディタに出會へてとても幸せです！この感じはかつて ed を數年使つた後に vi に出會つた時感じたものに似てゐます。今まで私が愛した ed や、まして vi を捨て去る日が來るとは考へもしませんでした…あぁ神樣、vi は偉大でした。でも今は、Vim。(Peter Jay Salzman, アメリカ合衆國)

こんな素晴らしいソフトウェアパッケージがあるなんて、私は本當に幸せです。どんな高價なソフトよりも良質で、しかも在庫切れといふことがありません。(Jeff Walker)

Vim のドキュメントを讀むといつも、エディタのパワーに昂奮し打ちのめされます。(William Edward Webber, オーストラリア)

Vim 萬歲！！それは vi のやうに "すぐ手の屆くところにあり"、vi がひどく必要とした擴張がなされてゐます: コマンドを實行すべきブロックの强調表示、簡單に操作でき消化できるヘルプ畫面、その他澤山。(Paul Pax)

私がこの素晴らしく便利なマクロをもう使はない理由は、今はそれらが組み込まれてゐる Vim を使つてゐるからです。(Stephen Riehm, ドイツ)

私は Vim のユーザーで、それを愛してゐます。自分の全てのプログラミング、C、C++、HTML、そのほか全てにおいて Vim を使用してゐます。(Tim Allwine)

私はオリジナルの vi と苦鬪すること數年の後に Vim を發見し、もうそれなしでは生きていけなくなりました。(Emmanuel Mogenet, アメリカ合衆國)

Emacs には Vim が活躍するほどに長く生き殘るチャンスはこれつぽつちもありません。加へて、これほどまでに詳細に渡つて記されたドキュメントを持つソフトウェアを見たことがありません---しかもそれは多くの商用ソフトウェアよりも素晴らしい！(Leiming Qian)

この Vim のバージョンは、それがどんなに素晴らしいものが發見するたびに人々を驚愕 させるでせう！(Tony Nugent, オーストラリア)

貴方のアドバイスに從ひ、つひに Vim を入手し、本當に感動しました。すぐに轉向しました。(Patrick Killelea, アメリカ合衆國)

Vim はとりわけ好きなシェアウェアで、特にバージョン 3.0 がお氣に入りです。これは本當にしつかりした仕事です。(Robert Colon, アメリカ合衆國)

Vim を使ふのは樂しいし、良く考へられていとても役に立つから、何故ほかの人たちがビジュアル開發ツールを使はうとするのか理解に苦しみます。Vim はパワフルで優雅で、一見單純ですが、實は 747 {譯注: ボーイングのジェット機} のやうに複雜で (とりわけ成長した私の .vimrc)、この素晴らしい仕事を續けて欲しいし、Vim はフリーソフトウェア界の中心です。(Louis-David Mitterand, アメリカ合衆國)

Vim を使ふことがこんなに素晴らしいなんて未だに信じられません。職場の仲閒は私がそれを譽めちぎるのを聞かされ過ぎて疲れてゐるでせう。他の人の目が輝き出します。(Rick Croote)

Emacs は起動と實行にかなり時閒がかかり、實際に使用するには大きくかさばつてゐて、インターフェイスは助けになるよりもむしろ混亂させます。しかしながら Vim は小さく、早く、パワフルで、素晴らしいインターフェイスを持つてゐて、あらゆる用途に使ふことができます。(Paal Ditlefsen)

Vim 3.0 を初めて手にした時から非常に熱狂しました。ほとんど何の問題點も見あたりませんでした。スワップファイルの扱ひとバックアップの可能性が非常に堅固で、ひとつのファイルを 2 重に編輯することから守つてもくれます。VI との互換性も非常に高いです (この數年私の腦はすつかり VI に洗腦されてゐるので、これは絕對條件です)。(Gert van Antwerpen, オランダ)

Vim のビジュアルモードは、凄くパワフルな機能です！(Tony Nugent, オーストラリア)

Vim は今までネットに存在しなかつた {譯注: 定冠詞 THE が付く} 唯一のもつとも偉大なソースコード作品と言はざるを得ないでせう。(Jim Battle, アメリカ合衆國)

實際、新しい vi が必要ならば Vim-3.0 をお奬めします。これは今まで見たなかで斷然最高のバージョンの vi です。(Albert W. Schueller)

Vim は良いエディタで他の如何なるものにも匹敵するものだ、といふべきでせう。(Ilya Beloozerov)

實を言へば時々 elvis、vile、xvi、calvin やその他のものを使つてゐました。だからこそ Vim が最高だと斷言できます！(Ferenc Deak, ハンガリー)

Vim は私が長年使用してきた斷然最高のエディタです。そして私が使用した全てのプラットフォームに存在するいろんなものをみてきました。それらのなかでも Vim は最高です。(Guy L. Oliver)

Vim は石ノミ以來、最高のエディタです。(Jose Unpingco, アメリカ合衆國)

かう言ひたい、Vim に出會つて私はつひに emacs から vi に鞍替へしました - あらゆる點で優れたエディタです: キーボードレイアウト、メモリ使用量、text alteration to name 3 {譯注: 譯不明}。(Mark Adam)

實際、今 vi 獨特の設定について知りたいのならば、Vim を起動してそのヘルプをチェックします！(Nikhil Patel, アメリカ合衆國)

vi ユーザーとして、Vim はテキストに關はる仕事を以前に遭遇してゐたよりも、はるかに愉快なものにしてくれました。(Steinar Knutsen, ノルウェー)

Vim をバージョン 3.0 から使つてゐます。そのころから、Solaris、Linux、そして OS/2 Warp で使用した唯一のエディタでした。友達全員に Vim を使ふやうに薦め、彼らはトライし、今も使ひ續けてゐます。Vim はインターネットからダウンロードした中で本當に最高のソフトウェアであり、知りうる限りで最高のエディタです。(Marco Eccettuato, イタリア)
