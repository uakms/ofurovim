Vim ユーザーの感想

*quotes.txt*    For Vim バージョン 8.1.  Last change: 2018 Mar 29


		  VIMリファレンスマニュアル    by Bram Moolenaar

ニュースやメールから集めた、Vimに關する素晴らしい引用を紹介する。

vim (vim) 名詞 - 溢れんばかりのバイタリティとエネルギー。 [ラテン語, 『力』の直接目的格, 力] (辭書)

Vim は vi より大變優れてゐるので、私の古い vi の多くの偉大な :map がすぐに廢れてしまつた！ (Tony Nugent, オーストラリア)

Windows の GUI 志向の强い影響で、Viを使ふ人々は實は人閒の姿をしたエイリアンなんぢやないかと思つてゐた。でも一度試してみるとそのパワーに病みつきになつて、今では Vim 以外のソフトウェアで Vim のやうにキータイプするまでになつてしまつた。これが、テキストを編輯するあらゆるアプリケーションにVimを埋め込まれて欲しいと願ふ理由。(José Fonseca)

私は 12 年來の emacs ユーザーだつたが、相次ぐ互換性のないバージョンアップや、役立たずの追加パッケージ群、一貫性のない操作方法などに嫌氣がさして 1 年前に Vim に乘り換へた。まさに生涯最高の選擇の1つだつた。 (Joel Burton)

資料の作成や修正に多くのプログラムを使用したが、編輯のほとんどは GNU-Linux (RedHat 4.2) 上の Vim のバージョン 4.5 と 5.0 で行はれた。(Arnold Robbins, イスラエル, "Learning the Vi editor" の著者)

私は幾つものオープンソフトを使ひ、さらに多くのソフトを見てきたが、その中でも Vim は大變使ひやすく、とても完成度が高く、linux kernel に次ぐ最高の作品だ。(Peter Jay Salzman)

SourceForge の「全體が」Vim とその PHP 用のイカシた構文强調を使つて書かれてゐる、といふことは大して重要ではない。大事なのは SF.net の技術スタッフ全員が Vim を使つてゐるといふことであり、そして貴方が Vim を使ふことを大いに歡迎してゐるといふことだ！(Tim Perdue)

Vim は私の七つ道具の 1 つで、代用品がありません。なんてすばらしい作品なんだらう！(Biju Chacko)

私の元カノは emacs に走りました。もちろん、彼女とはうまくいきませんでした。(Geoffrey Mann)

魚が水のことを意識しないと思はれるのと同樣に、私はほとんど Vim について意識しない。それは全ての中にある環境である。私は相當に忙しいシステム管理者で、多くの異なるプラットフォームを扱つてゐる。Vimは 私のシステム全てに共通して存在する唯一のもので、「時々クラッシュする」といふこともない唯一のものだ。新しいシステムに Vim が入つてゐなかつたときには、私は眞つ先にインストールする。どの環境でもまつたく信賴のおける同じツールを使ふことができることは素晴らしく、おかげで Vim については無視することができ、その他のものについては考へることができるのだ。(Pete Schaeffer)

最近ノキアの端末を通して telnet 經由で Vim を起動することに成功した、現在 Palm Pilot でも素晴らしく動いたことを報告できる。(Allan Kelly, スコットランド)

Bram! ’VIM’ といふなんてものすごい仕事をしたものだ！使へば使ふほど感動するよ (私は古い ’vi’ のダイハードで 1970 年代後半の初期の ’emacs’ から使ひ始め、1983 年に初期の UNIX で ’vi’ を見つけることで救はれた)。私の考へでは、標準のトップエディタとして ’VIM’ が ’emacs’ に置換はる時だよ。(Bo Thide’, スウェーデン)

私も Vim を深く愛し使つてゐる。(Larry Wall {譯注: Perlの作者})

Vi はまるでフェラーリだ。初心者にはまるで暴れ馬だが、いつたんコツさえ摑んでしまへば、非常に小さくパワフルかつ高速だ！(不明) Vim はまるでフェラーリの新モデルだ。撥音もそつくり - "VIIIIIIMMM!"。(Stephen Riehm, ドイツ)

Schon bei Nutzung eines Bruchteils der Vim-Funktionen wird der Benutzer recht schnell die Vorzuege dieses Editors kennen- und schaetzenlernen. 飜譯: Vim の機能をほんの少し使ふだけでも、ユーザーはすぐに使ふことができこのエディタの强みを正しく評價するだらう。(Garry Glendown, conclusion of an article on Vim in iX magazine 9/1998)

つい最近 Vi についてのオライリーの本を手に入れた (Vim についても深く言及されてゐた)。このアプリケーションがあまりにパワフルで驚愕した。(Jeffrey Rankin)

このガイドは Windows 9.x 上の gvim で書いてゐる。これは神が裸の女性を創造して以來の最高傑作なんぢやないだらうか。(Michael DiBernardo)

坊主、私は私自身 Vim についてほとんど全てのことを知つてゐると考へてゐた。しかしいつもオンラインドキュメントを眺めるたびに、それまで知らなかつた Vim の機能の小さな、しかしクールな面を見つけるんだ！これは私がみた中でも (多くの商業製品と比べてさへも) 最高のドキュメントであると言はねばならない。(Gautam Mudunuri)

Vim 4.5 は本當に素晴らしい(ファンタスティックな)エディタだ。Vim にはものすごぉぉぉぉい多くの機能があり更に重要なことには、初期設定が素晴らしいので本當に何の變更も無しに使ふことができるのだ！！Vim 作者への私の驚きと感謝の念はどんな言葉をもつても語り盡くせない。この先もさうであつて。(Vikas, アメリカ合衆國)

人々が他の Vi エディタを Vim のクローンと呼ぶやうになるまでどのくらゐかかるだらうか? (Darren Hiebert)

"Perfect Programmer’s Editor" のスレッドにある [make のエラーに基づいて自動的にファイルの位置へ飛ぶ] といふのを讀んで、Vim は既にそれをサポートしてゐるので嬉しかつた。(Brendan Macmillan, オーストラリア)

Vim (5.0) を發見して多くの知人とそれについて話をした。Vim は Vi の新世紀 (Millenium) を象徵するんだ、と彼らに言ひ聞かせた。本當にありがたう！(Matt F. Valentine)

"vi" は今から "Vim Imitation (Vim のものまね)" と呼ばれるべきで、逆ではないと考へる {譯注: Vim はもともと Vi IMitation に端を發してゐる}。(Rungun Ramanathan)

Vim の法則 {譯注: 數學の公式風に！}:
プログラム P の考へ得る限りの動作空閒 B の中で各々の要素を b とした時、プログラム P の全ユーザー空閒Uのうち少なくとも 1 人のユーザー u が、ある動作bを許された動作空閒 B’ に加へるやうに要求するまでにかかる時閒は有限である。(このとき B’ は B’ <= B となる) 言ひ換へれば: 遲かれ早かれ全員がオプションとして全てを欲しがるんだよ。(Negri)

あたらしい計算機プラットフォームに移る時はいつも、決まつてまづ最初にやるのは Vim を移植すること。後に configure を使つた時にコンパイルの簡單さにビックリした。(A.M. Sabuncu, トルコ)

オプションが本當に優秀でとてもパワフルだ。(Anish Maharaj)

ユーザーインターフェイスのデザインは春であり、80x24 テキスト專用モードのブティックからの「言葉は復讐に戾つた！」。Vi エディタのクローンである Vim は、5.0 のシンタックスハイライトによつて 3 月のデスクトップに眩暈のするパステルの景色をもたらした。金切り聲とカスタマイズ性、Vim は傳統的な Vi のシングルキーマクロ集に獨自の解釋をすることで、我々を驚かせる。{譯注: 原文が難解でまつたく自信無} http://www.ntk.net/index.cgi?back=archive98/now0327.txt&line=179#l

Vim 5 の衝擊をどうしても傳へたかつたんだ！シンタックスハイライティング: これ無しでどうやつて生きろって？！人類最高のエディタを作つてくれてありがたう！(Mun Johl, アメリカ合衆國)

重ね重ね Vim には感謝する。Linux で每日使つてゐる。(Eric Foster-Johnson, "UNIX Programming Tools"の著者)

史上最高のエディタ (Stuart Woolford)

Vim の素晴らしい機能のほとんどを少なくとも 1 度は、ものによつてはもう何度も頻繁に使つてきた。正直に白狀しよう、どれ 1 つ缺けても生きていけない。vi を使つてゐた頃の自分と生產性を比べると、優に倍にはなつてゐる。(Sitaram Chamarty)

Vim を愛してゐる。信じられないよ(incredible)。最初の子供に Vimberly と名づけた。(Jose Unpingco, アメリカ合衆國)

ヒント: "Vim" は "vi improved (vi改)" つていふほうが良い！(Sven Guckes, ドイツ)

每日 Vim を使つてゐる。他のどんなプログラムよりも Vim と共に過ごす時閒が長い…。存在する vi クローンの中で最高だ。とてもすごいよ。(Craig Sanders, オーストラリア)

その無限アンドゥ/リドゥで私を幾度となく悲しみから救つてくれた--- Vim を使用するやうに强く薦める。(Terry Brown)

私の考へうるなかで、この惑星で最良のテキストエディタを書いて(作成して)くれたことにたいへん感謝する。次に猫を飼ふときは "Vim" と命名したい。(Bob Sheehan, アメリカ合衆國)

:set all とタイプすればオプションで畫面が埋め盡くされる。畫面全體のものが設定か非設定かされてゐる。wrapmargin や modelines や showmode といつた古い友人たちも見つけたが、新しい友人たちで畫面は埋め盡くされてゐた。私は彼らを愛してゐる！！私は Vim を愛してゐる！！このエディタに出會へてとても幸せだ！この感じはかつて ed を數年使つた後に vi に出會つた時感じたものに似てゐる。今まで私が愛した ed や、まして vi を捨て去る日が來るとは考へもしなかつた…あぁ神樣、vi は偉大だつた。でも今は、Vim。(Peter Jay Salzman, アメリカ合衆國)

こんな素晴らしいソフトウェアパッケージがあるなんて、私は本當に幸せだ。どんな高價なソフトよりも良質で、しかも在庫切れといふことがない。(Jeff Walker)

Vim のドキュメントを讀むといつも、エディタのパワーに昂奮し打ちのめされる。(William Edward Webber, オーストラリア)

Vim 萬歲！！それは vi のやうに "すぐ手の屆くところにあり"、vi がひどく必要とした擴張がなされてゐる: コマンドを實行すべきブロックの强調表示、簡單に操作でき消化できるヘルプ畫面、その他澤山。(Paul Pax)

私がこの素晴らしく便利なマクロをもう使はない理由は、今はそれらが組み込まれてゐる Vim を使つてゐるからだ。(Stephen Riehm, ドイツ)

私は Vim のユーザーで、それを愛してゐる。自分の全てのプログラミング、C、C++、HTML、そのほか全てにおいて Vim を使用してゐる。(Tim Allwine)

私はオリジナルの vi と苦鬪すること數年の後に Vim を發見し、もうそれなしでは生きていけなくなつた。(Emmanuel Mogenet, アメリカ合衆國)

Emacs には Vim が活躍するほどに長く生き殘るチャンスはこれつぽつちもない。加へて、これほどまでに詳細に渡つて記されたドキュメントを持つソフトウェアを見たことがない—しかもそれは多くの商用ソフトウェアよりも素晴らしい！(Leiming Qian)

この Vim のバージョンは、それがどんなに素晴らしいものが發見するたびに人々を驚愕 させるだらう！(Tony Nugent, オーストラリア)

貴方のアドバイスに從ひ、つひに Vim を入手し、本當に感動した。すぐに轉向した。(Patrick Killelea, アメリカ合衆國)

Vim はとりわけ好きなシェアウェアで、特にバージョン 3.0 がお氣に入りだ。これは本當にしつかりした仕事だ。(Robert Colon, アメリカ合衆國)

Vim を使ふは樂しいし、良く考へられていとても役に立つから、何故ほかの人たちがビジュアル開發ツールを使はうとするのか理解に苦しむ。Vim はパワフルで優雅で、一見單純だが、實は 747 {譯注: ボーイングのジェット機} のやうに複雜で (とりわけ成長した私の .vimrc)、この素晴らしい仕事を續けて欲しいし、Vim はフリーソフトウェア界の中心だ。(Louis-David Mitterand, アメリカ合衆國)

Vim を使ふことがこんなに素晴らしいなんて未だに信じられない。職場の仲閒は私がそれを譽めちぎるのを聞かされ過ぎて疲れてゐるだらう。他の人の目が輝きだす。(Rick Croote)

Emacs は起動と實行にかなり時閒がかかり、實際に使用するには大きくかさばつてゐて、インターフェイスは助けになるよりもむしろ混亂させる。しかしながら Vim は小さく、早く、パワフルで、素晴らしいインターフェイスを持つてゐて、あらゆる用途に使ふことができる。(Paal Ditlefsen)

Vim 3.0 を初めて得た時から、非常に熱狂した。ほとんど何の問題點もみあたらなかつた。スワップファイルの扱ひとバックアップの可能性が非常に堅固で、ひとつのファイルを 2 重に編輯することから守つてもくれる。VI との互換性も非常に高い (この數年私の腦はすつかり VI に洗腦されてゐるので、これは絕對條件である)。(Gert van Antwerpen, オランダ)

Vim のビジュアルモードは、凄くパワフルな機能だ！(Tony Nugent, オーストラリア)

Vim は今までネットに存在しなかつた {譯注: 定冠詞 THE が付く} 唯一のもつとも偉大なソースコード作品と言はざるを得ないだらう。(Jim Battle, アメリカ合衆國)

實際、新しい vi が必要ならば Vim-3.0 をお奬めする。これは今まで見たなかで斷然最高のバージョンの vi だ。(Albert W. Schueller)

Vim は良いエディタで他の如何なるものにも匹敵するものだ、といふべきだらう。(Ilya Beloozerov)

實を言へば時々 elvis、vile、xvi、calvin やその他の使つてゐた。だからこそ Vim が最高だと斷言できる！(Ferenc Deak, ハンガリー)

Vim は私が長年使用してきた斷然最高のエディタである。そして私が使用した全てのプラットフォームに存在するいろんなものをみてきた。それらのなかでも Vim は最高だ。(Guy L. Oliver)

Vim は石ノミ以來、最高のエディタです。(Jose Unpingco, アメリカ合衆國)

かう言ひたい、Vim に出會つて私はつひに emacs から vi に鞍替へした - あらゆる點で優れたエディタだ: キーボードレイアウト、メモリ使用量、text alteration to name 3 {譯注: 譯不明}。(Mark Adam)

實際、今 vi 獨特の設定について知りたいのならば、Vim を起動してそのヘルプをチェックする！(Nikhil Patel, アメリカ合衆國)

vi ユーザーとして、Vim はテキストに關はる仕事を以前に遭遇してゐたよりも、はるかに愉快なものにしてくれた。(Steinar Knutsen, ノルウェー)

Vim をバージョン 3.0 から使つてゐる。そのころから、Solaris、Linux、そして OS/2Warp で使用した唯一のエディタだつた。友達全員に Vim を使ふやうに薦め、彼らはトライし、今も使ひ續けてゐる。Vim はインターネットからダウンロードした中で本當に最高のソフトウェアであり、知りうる限りで最高のエディタである。(Marco Eccettuato, イタリア)
